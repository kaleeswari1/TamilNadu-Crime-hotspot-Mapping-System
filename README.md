# 🛡️ Tamil Nadu Crime Hotspot Map

## 📌 Project Overview

The **Tamil Nadu Crime Hotspot Map** is an interactive web-based crime visualization system that displays crime hotspot information across selected districts of Tamil Nadu. The application helps users explore crime statistics through an interactive map, district-wise filtering, search functionality, and crime-type analysis.

The project is built using **HTML**, **CSS**, **JavaScript**, and the **Leaflet.js** mapping library with **OpenStreetMap** tiles.

---

# 🎯 Objectives

* Visualize crime hotspots on an interactive map.
* Display district-wise crime statistics.
* Filter crimes by different crime categories.
* Provide an easy-to-use interface for crime analysis.
* Improve understanding of crime distribution using geographical visualization.

---

# ✨ Features

* 🗺️ Interactive Tamil Nadu map

# 🔎 Working

1. Open the application.
2. Select a district from the sidebar or search bar.
3. The map automatically zooms to the selected district.
4. Crime hotspots are displayed using colored circles.
5. Click on any hotspot to view detailed crime information.
6. Apply crime-type filters to visualize specific crime categories.
7. The statistics panel updates dynamically based on the selected district.
